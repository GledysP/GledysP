## 👩‍💻 Sobre Mí

Soy una Ingeniera de Sistemas graduada en 2019 con un enfoque pragmático hacia el desarrollo de software. Me especializo en conectar el mundo del desarrollo (Angular 19, Java 21) con la infraestructura operativa (Docker, Linux, CI/CD). No solo escribo código; diseño soluciones que son seguras, desplegables y escalables desde el día uno.

* 🚀 **Actualmente construyendo:** [ReparaSuite](https://github.com/GledysP/reparasuite-root), un ecosistema SaaS con Backend Único y Frontend Multicliente.
* ☁️ **Me apasiona:** El Clean Code, Database Versioning (Flyway), la seguridad web (JWT, HttpOnly Cookies) y la automatización (GitHub Actions).
* 🌱 **Aprendiendo:** Estrategias avanzadas de Testing (E2E) y orquestación de despliegues en la nube (VPS/Linux).
